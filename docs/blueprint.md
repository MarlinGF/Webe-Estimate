# **App Name**: WeBeEstimate

## Core Features:

- Estimate Creation: Create and manage estimates with line items from service and parts libraries or custom entries.
- Invoice Conversion: Convert approved estimates to invoices with a single click, carrying over all estimate details.
- Payment Integration: Enable clients to pay invoices securely through Stripe and manage payment statuses.
- Library Management: Add, edit, and manage service and parts libraries with pricing templates and cost tracking.
- Client Management: Save client profiles, attach past invoices and estimates, and view job history.
- WeBe Project Integration: Connect directly to projects posted on WeBeFriends.com and send estimates directly to project posters.
- AI-Powered Smart Description Tool: Suggest item descriptions using AI based on keywords. This AI-powered tool helps business users quickly generate professional and detailed descriptions for line items.

## Style Guidelines:

- Primary color: #090165 (Blue) for a professional and trustworthy feel, aligning with the WeBe brand.
- Background color: #F6F6F9 (Light Gray) to provide a clean and modern backdrop.
- Accent color: #49E670 (Neon Green) to highlight key actions and important information.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design. Note: currently only Google Fonts are supported.
- Use rounded, clean icons in line with the Android/iOS hybrid card style to represent different services and actions.
- Implement a sidebar navigation on desktop and a hamburger menu on mobile for responsive design.
- Subtle animations and transitions to enhance user experience, such as loading indicators and smooth transitions between pages.
